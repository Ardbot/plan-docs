Формат для геопространственных данных.
Представляет собой sqlite базу данных. Может содержать несколько слоев (векторных и растровых).

По отзывам, возможны проблемы (потеря данных) при повреждении файла.
[MetaShape](../../soft/MetaShape.md) может сохранять ортофотопланы в gpkg формате (в виде пирамиды тайлов (ортомозаики)). Это позволяет значительно быстрее грузить растровые данные в [index](projects/soft/QGIS/index.md).
